# Changing Minds

Changing Minds is an AI-powered conversation strategy game where every word matters.

Solve human conflicts through conversation.

Convince a struggling student, defend an innocent client in court, interrogate suspects, negotiate impossible situations, and uncover the truth—one conversation at a time.

## Features

- AI-powered conversations
- Dynamic trust & frustration system
- Multiple conversation outcomes
- Case-based gameplay
- FastAPI backend
- Google Gemini integration

## Tech Stack

- HTML
- CSS
- JavaScript
- Python
- FastAPI
- Google Gemini
